# MAUNG THUN YA SSH Manager

A comprehensive SSH & VPN user management system for Ubuntu servers.

## Features
- User creation/removal
- User information display
- Port management
- SlowDNS management
- Telegram backup/restore
- Multiple protocol support (SSH, OpenVPN, Dropbear, Squid, etc.)

## Installation

### Quick Install (One Line)
```
bash <(curl -s https://raw.githubusercontent.com/zawtunwai/maungthunya-vip/main/pre_install.sh)
```
